Ejercicio 1
o Maquinas
Escribir una expresión utilizando funciones flecha y métodos de orden
superior (filter y reduce) para calcular el valor total de las máquinas
del inventario detallado a continuación:
Hint: Filter(), Reduce(), Arreglos unidimensionales, Objetos, Funciones flecha
Ejemplos:
const inventory = [
{type: "machine", value: 5000},
{type: "machine", value: 650},
{type: "duck", value: 10},
{type: "furniture", value: 1200},
{type: "machine", value: 77}
]

Ejercicio 2
o Copiando pares
Implementar una función CopyPairs que utilizando las novedades de ES6
tome como parámetro un arreglo de enteros. Luego la función deberá
recorrer los números del arreglo y para los que sean pares los copiara a

un nuevo arreglo. Al terminar de copiar todos los números pares deberá
retornar el nuevo arreglo como resultado.
Finalmente se deberá llamar a dicha función y luego imprimir cada número
del arreglo resultado en la consola del navegador.
Hint: Map(), Arreglos unidimensionales, Funciones flecha
Ejemplos:
CopyPairs([2,3,5,12,31]) => [2,12]
CopyPairs([10,18,191,13,248]) => [10,18,248]

Ejercicio 3
o Sumando matrices
Implementar una función rotString que tome como parámetro una cadena y
un número entero para luego alterar cada letra de la cadena avanzando o
retrocediendo dependiendo el signo del entero recibido como parámetro.
Para el avance y retroceso se tiene como referencia el alfabeto, y vale
mencionar que el mismo debe estar representado de forma circular para
evitar salir del mismo.
Hint: Operadores aritméticos, Map(), Arreglos unidimensionales, Funciones flecha
Ejemplos:
rotString("abcd", 1) ➞ "bcde"
rotString("abcd", -1) ➞ "zabc"

Ejercicio 4
o Cuadrados mágicos
Implementar una función magicSquare que tome como parámetro una matriz
cuadrada y determine si la misma representa un cuadrado mágico, en caso
afirmativo debe retornar True y en caso contrario False.
Un cuadrado mágico es una disposición de números naturales en una tabla cuadrada, de modo
que las sumas de cada columna, de cada fila y de cada diagonal son iguales.
Hint: Matrices, Map(), Funciones flecha
Ejemplos:
magicSquare(array([[3, 1, 5],[4, 7, 2],[9, 8, 6]])) => False
magicSquare(array([[2, 7, 6],[9, 5, 1],[4, 3, 8]])) => True

Ejercicio 5
o Factores primos
Implementar una función primeFactors que reciba como parámetro un número
entero positivo y retorne un arreglo de enteros conformado por los
factores primos del número recibido.
Hint: Operadores aritméticos, Map(), Arreglos unidimensionales, Funciones flecha
Ejemplos:
primeFactors(20) ➞ [2, 2, 5]
primeFactors(100) ➞ [2, 2, 5, 5]

Ejercicio 6
o Patrones repetidos
Codificar una función repeatPattern que tome como parámetros dos
cadenas, y retorna true en caso que la segunda cadena repita el patrón
de repetición de la primera.
Hint: Map(), Arreglos unidimensionales, Funciones flecha

Ejemplos:
repeatPattern("ABAB", "CDCD") ➞ true
repeatPattern("ABCBA", "BCDCB") ➞ true
repeatPattern("FFGG", "CDCD") ➞ false
repeatPattern("FFFF", "ABCD") ➞ false
